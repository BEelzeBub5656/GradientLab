# Gradient Lab · 渐变实验室

> 浙江树人学院 · 人工智能课程大作业 · Vibe Coding 开发实践

一个面向设计师和前端开发者的渐变色实时预览与生成工具。纯 HTML/CSS/JS 实现，浏览器直接打开运行。

## 功能

- 🎨 **渐变实时预览** — 支持线性/径向两种模式，角度可拖拽调节
- 🎯 **多颜色点编辑** — 自由添加、删除、调色
- 📚 **灵感预设库** — 8 组精选配色（日落、海洋、极光、霓虹、紫夜、森林、糖果、火红）
- 🎲 **智能随机生成** — 5 种 HSL 配色算法
- 📋 **一键复制 CSS** — 自动生成标准 CSS gradient 代码

## 使用方式

浏览器直接打开 `GradientLab.html` 即可运行。

## 开发方式

通过 Hermes Agent (DeepSeek v4 Pro) 以自然语言提示词驱动开发（Vibe Coding），配合 Playwright 浏览器自动化进行测试验证。
